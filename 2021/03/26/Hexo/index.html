Hexo博客搭建教程

准备工作

  • 确保电脑里已经安装好新版的 node.js

安装 Hexo

创建博客项目

  1. 在创建博客项目的文件夹执行 hexo init <项目名称>
  1. 进入到 blog 目录下 cd blog，建立第一篇文章 hexo new <文章名称>
  1. 运行 hexo 服务

可以在本地访问你的博客网站了，默认端口号是 4000

打包部署

  1. 打包
    直接运行打包命令，这样就能使用 Hexo 引擎将 Markdown 格式的文件解析成可以使用浏览器查看的 HTML 文件，HTML 文件存储在 public 目录下
